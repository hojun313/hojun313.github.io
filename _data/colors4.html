Further Consolidated Color Palette Demo

This page demonstrates the further consolidated color palette, with project card and blog post backgrounds integrated into a single shade.

Background Colors

These are the primary background colors used throughout the site.

--bg-body (#121212) - Main Body Background
--bg-container (#1e1e1e) - General Container Background / Even Table Rows
--bg-card-post (#2a2a32) - Project Card, Blog Post, Table Header, Odd Table Row, Inline Code Background
--bg-black (#000000) - Preformatted Code Block Background

Text Colors

Demonstration of various text colors on a typical container background.

--text-heading (#FFFFFF) - Main Titles / Strong Emphasis

--text-primary (#E0E0E0) - Default Body Text / Blog Summaries

--text-secondary (#bdbdbd) - Secondary Text / Descriptions / Pagination Numbers

--accent-main (#BB86FC) - Accent Text / Links

Main Accent: --accent-main (#BB86FC)

This color is used for primary interactive elements and highlights.
