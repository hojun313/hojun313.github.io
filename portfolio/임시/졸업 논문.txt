프로젝트 명: 현실과 가상을 잇는 FPS: 컴퓨터 비전 기반 총기 인식 및 햅틱 피드백 시스템

1. 프로젝트 개요
오픈소스 FPS 게임 '어썰트 큐브'에 컴퓨터 비전 기술과 하드웨어 제어를 접목, 플레이어가 현실에서 사용하는 총기 모델을 게임이 실시간으로 인식하고 격발 시 물리적인 반동(햅틱 피드백)을 제공하는 시스템 개발. 기존 키보드/마우스 환경을 넘어선 차세대 FPS 게임의 몰입감 구현을 목표로 함.

2. 주요 기능 및 구현 내용
* 실시간 총기 인식 모듈 (Python, OpenCV)
    * 웹캠을 통해 플레이어의 총기 모델(권총, 라이플 등)을 실시간으로 이미지 분석.
    * OpenCV를 활용, 특정 총기의 특징점을 학습하여 95% 이상의 정확도로 식별.
    * 인식된 총기 정보를 게임 엔진에 전달하여 캐릭터의 무기를 자동 교체.

* 게임-하드웨어 양방향 통신 시스템
    * (총기 인식 → 게임) Python 총기 인식 시스템의 결과를 C++ 게임 엔진으로 전송하는 소켓 통신 파이프라인 구축.
    * (게임 → 햅틱) 게임 내 '격발' 이벤트 발생 시, 외부 하드웨어(솔레노이드)로 신호를 보내도록 게임 소스 코드 수정.

* 물리적 반동을 위한 햅틱 피드백 하드웨어
    * 총기 모델에 '솔레노이드'를 장착, 게임의 격발 신호에 맞춰 물리적 충격을 발생시키는 햅틱 장치 제작.
    * 총기 종류별로 다른 타격감을 손으로 직접 느낄 수 있는 플레이 경험 제공.

* 게임 엔진 커스터마이징 (C++, 어썰트 큐브 소스 코드)
    * weapon.cpp, weapon.h 등 엔진 핵심 소스 코드 분석 및 수정.
    * 외부 장치와의 통신을 위한 무기 변경 및 격발 이벤트 송수신 인터페이스 구현.

3. 본인의 역할 및 기여
* 전체 시스템 아키텍처 설계
    * Python 비전 시스템, C++ 게임 엔진, 외부 하드웨어를 연동하는 전체 시스템 구조 설계.
* 핵심 로직 개발 및 구현
    * C++ 게임 엔진의 무기 소스 코드를 직접 수정하여, 외부 신호에 따른 무기 교체 및 격발 이벤트 송출 기능 구현.
* 하드웨어 프로토타이핑 및 안정화
    * 솔레노이드 기반 햅틱 피드백 장치 제작.
    * 소프트웨어-하드웨어 간 신호 전달 과정의 문제를 해결하고 시스템 안정화 주도.
* 프로젝트 관리 및 기술 문서 작성
    * 팀 개발 일정 조율 및 진행 관리.
    * 기초조사서, 최종보고서, 발표자료 등 프로젝트 전 과정에 대한 기술 문서 작성.

4. 기술 스택
* 언어: C++, Python
* 게임 엔진: 어썰트 큐브 (Cube Engine 기반)
* 라이브러리: OpenCV
* 하드웨어: 솔레노이드, (아두이노/라즈베리파이 등 제어 보드)

5. 성과 및 의의
* 소프트웨어와 하드웨어 결합을 통해 FPS 게임의 몰입감을 극대화하는 프로토타입 완성.
* 오픈소스 게임 엔진 직접 수정 경험을 통해 복잡한 상용 엔진 환경에 대한 빠른 적응력 및 기여 가능성 확보.
